- Static webserver: [http-server][httpserver]

Other tools:
- Shell: [fish][fish]
- Code editor: [vim][vim] / [neovim][nvim]
- Time sheet: [timetrap][timetrap]
- Local Tunnel: [localtunnel.me][localtunnel]
- SSL: [Lets Encrypt][letsencrypt]
- Static host: [github pages][ghpages]
